## 🛠️ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- (Optional) A local server for development — e.g., [VS Code Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/ahmadaizazdev/Elevate.git

# 2. Navigate into the project folder
cd Elevate

# 3. Open index.html in your browser
#    (or use Live Server for a better experience)
```

### Usage

1. **Choose a document type** – Click **CV** or **Resume** on the landing page.
2. **Pick a template** – Select from Classic, Minimal, or Executive.
3. **Edit your content** – Use the sidebar to add, remove, or reorder sections. Click any section to edit its title or body directly.
4. **Adjust formatting** – Use the toolbar for bold, italic, lists, and alignment.
5. **Customise margins & spacing** – Choose Narrow, Normal, or Wide margins, and Compact, Normal, or Relaxed spacing.
6. **Export to PDF** – Click the **Export PDF** button and download your polished, A4‑aligned document.
